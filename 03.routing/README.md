### Install Dependency
1. react-router: 6.0.x(2021.6 beta)      : <Routes>, <Route>
2. react-router-dom: 6.0.x(2021.6 beta)  : <HashRouter>, <BrowserRouter>

```bash
$ npm i react-router react-router-dom
```

### Run Examples
```bash
$ npm run dev [src=01|02|03|04|05|06|..]
```
